## 프로젝트 소개

- E-mail과 소셜로 회원가입을 진행하고 로그인을 할 수 있는 페이지입니다.
- 로그인이 완료되면 Mypage로 이동이 되어 유저정보를 확인하고 변경 할 수 있습니다.
